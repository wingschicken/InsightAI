About InsightAI

InsightAI is a FastAPI-based security scanning web app with non-admin-safe Nmap profiles, optional web security discovery, AI-powered analysis, chat with RAG support, and operational status monitoring.

The system keeps scanning safe for unprivileged containers by using TCP connect-style Nmap scans rather than raw packet scans. It also provides a separate RAG backend to ingest knowledge and answer context-aware security questions.

Available APIs

Use these endpoints to interact with InsightAI programmatically:

  • GET /ping — health check for the web app.
  • GET /api/status — system health including CPU, memory, disk, and RAG service reachability.
  • POST /api/scan — run a scan.
  • POST /api/rag/ingest — trigger the RAG service to ingest its knowledge file.
  • POST /api/rag_chat/{chat_id} — send a chat message via RAG and save the conversation.

Example scan request:

{
  "target": "scanme.nmap.org",
  "profile": "basic",
  "web_scan": true
}

Example RAG ingest call:

POST /api/rag/ingest

Profiles

All supported scan profiles use safe, non-admin Nmap options:

  • fast_recon
  • basic
  • top_ports
  • service
  • full
  • nonadmin
