About the Security Scanner

This container runs a single FastAPI application that performs Nmap network scanning, web header checks, OWASP-style mapping, and AI-powered result analysis.

Scans inside Docker may be slower than on your host PC because Docker networking and the container runtime add a bit of overhead. The app uses safer, faster Nmap settings by default to keep scans practical inside the container.

How the API works

Use the endpoints below for automated access.

GET /ping
POST /api/scan

The scan endpoint expects JSON like:

{
  "target": "example.com",
  "profile": "basic",
  "web_scan": true
}

And returns structured JSON with network findings, web checks, OWASP mapping, and AI analysis.
